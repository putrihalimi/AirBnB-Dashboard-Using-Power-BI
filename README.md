# 📊 Data Visualization – Empowering Business with Effective Insights

## 📌 Project Overview

Project ini merupakan simulasi analisis data bisnis menggunakan **Power BI** untuk menghasilkan insight yang mendukung pengambilan keputusan strategis. Studi kasus ini terinspirasi dari praktik analitik bisnis yang digunakan oleh Tata Insights and Quants dalam mengubah data menjadi actionable insights.

Dashboard dikembangkan untuk:

* Menganalisis tren performa bulanan
* Mengidentifikasi pola pertumbuhan dan penurunan
* Menyajikan insight secara visual dan interaktif
* Mendukung data-driven decision making

---

## 🎯 Objectives

* Membersihkan dan mentransformasi data mentah
* Membangun model data yang terstruktur
* Membuat visualisasi interaktif untuk monitoring performa bisnis
* Menghasilkan insight yang mudah dipahami oleh stakeholder non-teknis

---

## 🛠 Tools & Technologies

* **Microsoft Power BI Desktop**
* Data Cleaning & Transformation (Power Query)
* Data Modeling
* DAX (Data Analysis Expressions)

---

## 📈 Key Features of Dashboard

* 📅 Analisis performa berdasarkan bulan (January – July)
* 📊 Perbandingan metrik antar periode
* 📌 Visual KPI summary
* 📉 Trend analysis untuk melihat pola pertumbuhan
* 🎛 Interactive filtering untuk eksplorasi data lebih lanjut

---

## 🔎 Insight Highlights

Beberapa insight yang dapat diperoleh dari dashboard:

* Identifikasi bulan dengan performa tertinggi dan terendah
* Pola fluktuasi data dari awal hingga pertengahan tahun
* Indikasi peningkatan atau penurunan performa secara bertahap
* Area yang memerlukan evaluasi atau strategi perbaikan

---

## 💡 Business Impact

Dashboard ini membantu:

* Mempercepat proses analisis data
* Mengurangi ketergantungan pada laporan manual
* Memberikan visual yang lebih komunikatif kepada manajemen
* Mendukung pengambilan keputusan berbasis data

Project ini menunjukkan kemampuan saya dalam:

* Data visualization best practices

* Membangun dashboard yang user-friendly
